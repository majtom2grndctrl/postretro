Packaging notes (PostRetro):
* Sourced as a single `Knight.glb` from the upstream repository at
  addons/kaykit_character_pack_adventures/Characters/gltf/Knight.glb (main branch).
* Converted GLB -> external glTF layout (scene.gltf + scene.bin + textures/*.png)
  with @gltf-transform (NodeIO) to ride the engine's existing external-glTF model
  pipeline (mirrors content/dev/models/decraniated_low_poly_retro_pixel/).
* The pack ships ~76 animation clips; the clip set was PRUNED to the four used by
  the reference enemy to keep the committed binary small:
    - Idle
    - Walking_A
    - 1H_Melee_Attack_Slice_Horizontal
    - Death_A
* Geometry, skeleton, and textures were NOT altered. Pruning removed only the
  unused animation clips and their now-orphaned keyframe accessors; the base-color
  texture was relocated into textures/ and the glTF JSON was minified.
